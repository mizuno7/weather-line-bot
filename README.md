傘通知bot
====

Overview

## Description
降水確率50%を超える時間帯がある場合、傘を持っていくことを通知するlinebotです
<br/>(傘が必要な場合のみ朝に通知が来ます)。

気象庁のデータをメタデータ化した情報を使用し、東京の天気情報を取得しています。
<br/>その他色々なコメントに対して反応します。
